Day 01
🐋 Whale Wallet Tracker
Query Dashboard
What To Do
  • Search "whale tracker" on Dune Discover tab
  • Fork a query that tracks wallets with 1000+ ETH
  • Save with your own title and run it
  • Create a new Dashboard → add this query
What You Learn

How to spot accumulation phases before price moves. When whales buy quietly, price usually follows. This is how smart traders get early entries.

Day 02
📊 DEX Volume Analysis
Query Contract
What To Do
  • Go to Settings → Contracts → Add a DEX contract
  • Find Uniswap or GMX contract on Arbiscan
  • Submit it as a decoding request
  • Write a query: daily volume of any token on DEX
What You Learn

How to distinguish real vs wash trading volume. Fake pumps have volume spikes with no new wallets. Real volume has consistent growth across many unique traders.

Day 03
👥 Token Holder Concentration
Query Dashboard
What To Do
  • Fork a "token holders" query from Discover
  • Modify it for a token you're trading or watching
  • Add a pie chart visualization to your Dashboard
  • Run and analyze top 10 holder percentages
What You Learn

If 3 wallets hold 70%+ of a token, it's a rug risk. Healthy tokens have distributed ownership. This saves you from losing money on concentrated dumps.

Day 04
💰 Smart Money Wallets
Query Dashboard
What To Do
  • Search "smart money" or "profitable wallets" on Discover
  • Fork a query tracking consistent profit wallets
  • Add a table widget to your Dashboard
  • Label and save 3 smart wallets you want to follow
What You Learn

How to copy-trade on-chain without a tool. These wallets bought tokens 2 weeks before 10x moves. Following them gives you a real trading edge.

Day 05
🔍 New Protocol Traction
Query Contract
What To Do
  • Find a newly deployed contract on Arbiscan (last 30 days)
  • Submit it as a decoding request on Dune
  • Write a query: transaction count over time for new contracts
  • Compare 3 new protocols side by side
What You Learn

How to spot trends before Twitter does. Rising transaction counts on a new contract = organic growth happening. This is the earliest possible signal.

Day 06
🏦 Liquidity Pool Monitor
Query Dashboard Contract
What To Do
  • Add a liquidity pool contract from any DEX
  • Fork a "liquidity depth" query from Discover
  • Add a line chart to your Dashboard showing LP changes
  • Track if liquidity is being added or removed over 7 days
What You Learn

Sudden liquidity removal = rug warning. Steady LP growth means developer confidence. This is one of the most useful signals for DeFi token safety.

Day 07
🚀 Build Your Master Dashboard
Query Dashboard Contract
What To Do
  • Create one final Dashboard that combines ALL 6 days of work
  • Add text widgets explaining what each section shows
  • Arrange: Whales → Volume → Holders → Smart Money → New Protocols → LP Health
  • Publish it publicly — this becomes your on-chain research page
  • Post the link on Twitter/X tagging @DuneAnalytics
What You Achieve

You now have a complete on-chain research workflow. Every token you consider trading, run through this dashboard. Whales buying + real volume + distributed holders + LP growing = strong buy signal. This is how professional analysts think.
